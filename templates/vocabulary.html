Точка входа в программирование Python Словарь Циклы Функции
Эта страница является словарем, который ты всегда сможешь открыть через верхнее меню и напомнить себе что значит то или иное определение.
  • Оператор - операция, выполняющая сравнение на равенство (==), на превосходство одного значения перед другогим(>, < или >=, <=), так же операция ИЛИ на Python = or, операция И = and, операция НЕ = not. *Будьте внимательны: операция '=' означает присваивание значения переменной, а операция '=='' лишь сравнивает их.
  • Комментарий - блок кода, начинающийся после знака решетки(#) и не играющий никакой роли в коде проекта, кроме пояснительной. 
					if 10 == 10:
						a = True
						# переменной а задано значение True, так как проверка на равенство прошла успешно
					print(a)
				
					>>> True
  • Псевдокод - код, похожий на настоящий, но приведенный лишь для примера в целях обучения.
  • Переменные - это 'коробки', в которые мы складываем информацию, воспользоваться которыми можно в любой части нашего кода. 
					a = 10 # число(int)
					b = 'привет' # строка (str)
					c = True # булева логика(bool(True/False))
					# в комментариях я описал типы данных, которые содержат наши переменнные a, b и c
					print(a, b, c)
				
					>>> 10 привет True
  • Типы данных: всего типов данных ссуществует много, но вот основные из них:
    1. int - целое число(7)
    2. float - число с плавающей точкой(7.234)
    3. bool - булева логика(True или False)
    4. str - строка('это строка')
  • Блок - специальное место в коде, отделенное двоеточием (:);
    если мы создаем функцию, цикл или условие, они обязательно должны содержать в себе блок кода после двоеточия!
